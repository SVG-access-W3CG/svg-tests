Basic inline SVG title test

Pass criteria:

Inline SVG root-element 'title' element (non-visible metadata)
Navigation: UA can navigate to root element 'title' element
Focus indicator: UA draws a focus indicator around the entire SVG element
Voicing: UA reads the text of the root element 'title' element ("This in an inline SVG file")
Inline SVG child element 'title' element (non-visible metadata)
Navigation: UA can navigate to 'title' element child of 'rect' element
Focus indicator: UA draws a focus indicator around the 'rect' element
Voicing: UA reads the text of the 'title' element child of 'rect' element ("This in a rect element")
